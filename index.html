Computer Lab Assets management system


Computer Lab Assets management system is a mini PHP project with CRUD operations. It is designed for a management of Computer department labs, Desktop, Laptop, mini pcs and their peripherals, Printers, Scanners, Installed softwares.

  • Frontend : Bootstrap
  • Middleware : Basic php
  • Backend : Mysql database
  • Test/run on : Windows Xampp cpanel 3.2.2
